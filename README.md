# lyric-translator

A Spotify web player that displays translated and romanized song lyrics.

**Live Demo:** https://lyric-translator.vercel.app/

## Built With

### Frontend

-   React

### Backend

-   Node.js
-   MongoDB

### Authorization

-   Spotify OAuth 2.0
-   Google OAuth 2.0

### Deployment

-   Vercel (client)
-   Render (server)
